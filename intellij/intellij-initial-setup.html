Initial Setup of IntelliJ IDEA

1. From the software center (type "software center" into the Windows search box), install:

IntelliJ IDEA

Java SE Development Kit 8 Update 141

Java 8 Update 141

2. Go to the website of IntelliJ and register for an account:

https://www.jetbrains.com/shop/eform/students

(YOU MUST USE YOUR AB-TECH EMAIL ADDRESS)

3. When you run IntelliJ IDEA for the first time, use your account info from step 2 to register it.

NOTE: If the installer doesn't create a shortcut on your desktop for IntelliJ IDEA, go to C:\Program Files (x86)\JetBrains\IntelliJ IDEA 2017.2.2\bin. Right click on idea64.exe, and select Send to, then Desktop (create shortcut). You can also add it to the task bar if you wish.

4. In IntelliJ IDEA, click File, Other Settings, then Default Project Structure.

5. Click SDKs, then click the green + sign.

6. Click JDK, and then navigate to C:\Program Files\Java\jdk1.8.0_141 and click OK.

7. On the left hand navigation bar, click Project.

8. Select 1.8 from the Project SDK drop down menu, then select 8 - Lambdas, type annotations... from the language level drop down. Click OK.

9. Click File, Other Settings, Default Settings.

10. Click Editor, then File and Code Templates.

11. With the Files tab selected, click Class.

12. In the top right window (above all other code), paste this:

/*
 * Documentation of help received:
 */

13. Replace the class declaration and everything below it with this:

public class ${NAME} {

    //CONSTANTS
    
    //IVARS
    
    //CONSTRUCTORS
    
    //GETTERS
    
    //SETTERS
    
    //OTHER METHODS

} //END OF CLASS

14. Click OK.
